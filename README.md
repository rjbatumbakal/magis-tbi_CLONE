# ADNU MAGIS Technology Business Incubator

Official website for the ADNU MAGIS Technology Business Incubator (TBI).

**MAGIS** - Maximizing Alliances for a Greater Innovative Society

A university-owned and operated technology business incubator serving technology-focused or technology-utilizing growth-oriented startups.

## Getting Started

### Local Development

1. Open `MaigsTBI_CLONE/magis_clone.html` in your web browser
2. Or use a local server:
   ```bash
   # Using Python 3
   python -m http.server 8000
   
   # Using Node.js (http-server)
   npx http-server
   ```

### Vercel Deployment

This project is ready for Vercel deployment:

1. Push to GitHub
2. Connect repository to Vercel
3. Vercel will automatically detect and deploy the static site
